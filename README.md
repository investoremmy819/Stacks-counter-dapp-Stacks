# Stacks Counter DApp

A simple decentralized app on the Stacks blockchain that demonstrates:
- Writing a Clarity smart contract
- Connecting a wallet with @stacks/connect
- Making contract calls with @stacks/transactions

## Features
- Increment a counter stored on-chain
- Read the current counter value
- Connect Stacks wallet

## Tech Stack
- Clarity
- JavaScript
- @stacks/connect
- @stacks/transactions

## Project Structure

contracts/
  counter.clar
frontend/
  index.html
  app.js

## Setup
1. Install Hiro Stacks Wallet
2. Deploy contract to Stacks Testnet
3. Open frontend/index.html
4. Connect wallet and interact with the contract
